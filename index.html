Factorio Map Share

Factorio is a great game where players are tasked with building a logistical dreamland on an alien planet. The game tasks players to creatively solve problems. But why do that when you can just download another player's hard work? Using Rails, players can log into this app to share and download save games. The coolest feature of this app is the screenshots created when a map is uploaded. Using the Ruby Gems paperclip and rubyzip, this app will automatically take the preview.png embeded in a map's zip file and display it on the map's download page.

  • Ruby
  • Ruby on Rails

Hand-Off

Did you know that when doctors change shifts, patient information is still passed back and forth on clipboards? Luckily for our group project we had a medical resident on our team who had a vision of a better way. Using a Rails backend, and an Angular front end, we developed a website that allows physicians to read, write, and edit a patient's chart online. Complete with a visual readout of lab results, and API calls to fetch diagnosis information, our app was designed to keep doctors informed of their patient's status, without the outdated clipboards.

  • Angular
  • Ruby on Rails

Rocket Beasts

I used Phaser.io and canvas to create a game similar to the Amiga classic Tanx. Then, to take my project one step further, I built a multiplayer server using NodeJS and websockets. The end result is a game where you can launch rockets at other live players, in real-time, all in your web browser.
